# Papers — MAM5020F 2026: Generative AI for Research

This folder collects the academic papers cited across the course. Each `Week-N/` folder contains:

- **PDFs** of openly-accessible papers (arXiv, open-access journals)
- **`MANIFEST.md`** — a list of every academic reference for that week, mapped to the sub-lesson it appears in. Includes paywalled papers (which can't be redistributed here) with their DOI and a UCT-library lookup link.

## Scope

- **In scope:** academic papers — arXiv preprints, journal articles with DOIs, conference papers (NeurIPS, FAccT, CHI, etc.).
- **Out of scope:** blog posts, tool documentation, books, news articles, software repositories. Those remain linked in-line on the lesson pages.

## Naming convention

`FirstAuthor_Year_ShortTitle.pdf`

Examples:
- `Magesh_2024_Hallucination-Free.pdf`
- `Chelli_2024_Hallucinated-Citations.pdf`
- `Liu_2024_Lost-in-the-Middle.pdf`

## Paywalled papers

Where a paper is paywalled and we can't distribute it openly, the `MANIFEST.md` for that week includes:
- Title, authors, year
- DOI / journal link
- A hint about UCT institutional access

Most paywalled papers can be retrieved via the UCT library's discovery service when you're on the campus network or signed in to the UCT VPN.
